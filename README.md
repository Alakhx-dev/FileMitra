# 🚀 File Mitra – Smart File Assistant

🌐 Live Demo: https://filemitraa.vercel.app/

File Mitra is a modern, fast, and stylish web-based file utility tool that allows users to convert, merge, compress, and optimize files directly in the browser with a premium UI experience.

---

## ✨ Features

* 📄 Convert files (PDF ↔ Word, Image ↔ PDF, etc.)
* 🔗 Merge multiple files into one
* 🗜️ Compress images and PDFs
* 🖼️ Image tools (resize, crop, optimize)
* ⚡ Fast client-side processing (no upload required)
* 🌙 Dark / Light theme toggle with smooth animation
* 🎨 Premium UI (glassmorphism + gradients + animations)
* 📱 Fully responsive (mobile + desktop)

---

## 🧠 Tech Stack

* HTML, CSS, JavaScript (Vanilla)
* PDF.js (for PDF rendering)
* Canvas API (for image processing)
* Vercel (deployment)

---

## 🚀 How It Works

* Files are processed directly in your browser
* No data is uploaded to any server
* Uses canvas-based compression for images and PDFs
* Fast and secure client-side execution

---

## ⚠️ Limitations

* PDF compression is browser-based (may not reduce size for already optimized PDFs)
* Large files may take more processing time
* Advanced compression (like SmallPDF) requires backend processing
